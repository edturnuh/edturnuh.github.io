
  # Portfolio v2

  This is a code bundle for Portfolio v2. The original project is available at https://www.figma.com/design/z7NRxNmlp2asUGc7eh3Qfc/Portfolio-v2.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deployment

  This project is configured to build for the root path `/`.

  The GitHub Actions workflow in `.github/workflows/deploy.yml` is set up to build and deploy this repo directly to GitHub Pages on pushes to `main`.

  For root-domain hosting, the GitHub repository itself should be named `edturnuh.github.io`.
