Design a minimal, typography-led personal portfolio homepage inspired by modern developer portfolios like confident-hand-921053.framer.app.

The layout should feel clean, calm, spacious, and confident. Prioritize strong typography hierarchy, generous whitespace, and simple structure over heavy visuals. No gradients, no flashy UI, no complex components. The site should feel intentional, product-focused, and fast.

Style direction:

Light theme (white background)

Black or near-black text

Subtle neutral greys for secondary text and dividers

One subtle accent color (muted blue or deep green)

System font or clean sans-serif (e.g. Inter-style)

Large margins and vertical spacing

Thin divider lines between sections

Very minimal UI elements

Page structure:

Top Navigation (simple text-based)
Left: Name (e.g. “Edward Turner”)
Right: Text links: Projects / Notes / About / Contact
No button styling, just clean inline links.

Hero Section
Large heading:
“Hey, I’m Edward — I build high-performing web products and growth experiments.”

Subheading paragraph:
Short 2–3 sentence description positioning me as a marketer, web developer, and product thinker who ships fast and focuses on results.

Optional small line beneath:
“Currently open to product, growth and web collaborations.”

Spacing should feel airy and confident.

Featured Projects Section
Section label in small uppercase or muted text: “Selected Projects”

Display projects as a clean vertical list (not cards).
Each row should include:

Year (left column, subtle)

Category (e.g. Growth / Web / Experiment)

Project Title (bold)

Small result tag (e.g. +18% conversion, 3-day build, 1st place hackathon)

Each row separated by thin divider lines.
Hover state should subtly change text color or underline title.

Writing / Notes Section
Section label: “Latest Notes”

Simple vertical list:

Title

Short one-line summary

Date in small muted text

Keep this minimal and text-focused.

About Preview Section
Small portrait placeholder (simple circular or square image).
Short paragraph about me — product-minded builder blending marketing, development, and experimentation.
Link: “Read more →”

Call to Action Section
Large but simple:
“Let’s build something interesting.”
Short line below:
Email address and/or “Get in touch →”

No heavy buttons. Keep it elegant.

Layout constraints:

Max width around 720–900px centered

Strong vertical rhythm

Consistent spacing scale

Mobile-friendly stacked layout

Tone:
Professional but not corporate.
Modern but not trendy.
Confident without being flashy.
Feels like someone who understands product, systems, and performance.

Do not include:

Large hero images

Background graphics

Animated elements

Gradient backgrounds

Overly stylized UI components

Focus on clarity, typography, spacing, and structure.
